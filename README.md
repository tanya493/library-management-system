# 📚 Library Management System

A console-based **Library Management System** built in Java as a PBL (Project-Based Learning) project.

---

## 🚀 Features

- 📖 **Book Management** — Add, Search, Update, Remove books
- 👤 **Member Management** — Register members (Student/Faculty/General)
- 🔄 **Issue & Return** — Borrow books with due dates, return with fine calculation
- 💰 **Fine System** — Auto fine of ₹2/day for overdue books
- 📋 **Reports** — View active borrows, overdue books, transaction history
- 📊 **Dashboard** — Live stats: total books, members, fines collected

---

## 🛠️ How to Run

```bash
javac LibraryManagementSystem.java
java LibraryManagementSystem
```

---

## 🏗️ Project Structure

| Class | Description |
|---|---|
| `Book` | Book data and copy tracking |
| `Member` | Member info and membership type |
| `Transaction` | Borrow/return with fine calculation |
| `Library` | Core business logic |
| `LibraryManagementSystem` | Main menu and console UI |

---

## 💻 Tech Used

- Java (Core)
- OOP Concepts — Classes, Encapsulation, Collections
- Java Collections — HashMap, ArrayList
- Java Time API — LocalDate for due dates
